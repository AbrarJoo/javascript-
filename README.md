Welcome to my JavaScript repository! This repository serves as a collection of my JavaScript projects, practice scripts, algorithms, and core concepts implemented using modern ES6+ syntax.

## 🚀 Purpose

The goal of this repository is to:

- Practice core JavaScript principles (closures, promises, async/await, DOM manipulation).
- Build practical, real-world utility mini-projects.
- Document my coding journey and progress in mastering JavaScript.

## 📂 Project Structure

Here is an overview of how the repository is organized:

```text
├── 01-basics/             # Fundamental concepts (variables, data types, loops)
├── 02-dom-manipulation/   # Interactive web elements and event listeners
├── 03-async-javascript/   # Promises, Fetch API, and Async/Await examples
```

💻 Getting Started
Prerequisites

To run the scripts or mini-projects locally, make sure you have a web browser (like Chrome, Firefox, or Edge) or Node.js installed if you want to run code in your terminal.
Installation & Setup

    Clone the repository:
    Bash

    git clone [https://github.com/AbrarJoo/javascript-.git](https://github.com/AbrarJoo/javascript-.git)

    Navigate into the directory:
    Bash

    cd javascript-

    Run a script (via Node.js):
    Bash

    node path/to/script.js

    (For UI/web projects, simply open the corresponding index.html file in your preferred browser).
